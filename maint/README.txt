After you have openned the set_website_to_and_from_maintenance directory, do the following:



(1) Open the file named set_website_to_and_from_maintenance/config.php and fill in your database connection settings

(2) Open the file named set_website_to_and_from_maintenance/TABLE.sql, copy the query to your server and
    run it to create the required table for this system

(3) Upload the directory named set_website_to_and_from_maintenance and its entire content to your server, location this directory via
    your web browser to see the system and you are done.
